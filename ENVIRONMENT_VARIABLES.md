# Required Environment Variables

## Critical Variables (Must Set)

### 1. **SECRET_KEY** ⚠️ CRITICAL
- **What it is**: Django's secret key for cryptographic signing
- **Why needed**: Security - used for sessions, CSRF tokens, password hashing
- **How to generate**:
  ```bash
  python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"
  ```
- **Value**: A long random string (Render can auto-generate this)
- **Status**: ✅ Should be set (Render auto-generates if using `generateValue: true`)

## Optional Variables

### 6. **PERSISTENT_DISK_PATH** (Optional)
- **What it is**: Path for persistent disk storage
- **Value**: `/var/data` (if using persistent disk)
- **When needed**: Only if you're using Render's persistent disk feature
- **Status**: Optional (but recommended for SQLite)

### 7. **DATABASE_URL** (Optional)
- **What it is**: PostgreSQL connection string
- **When needed**: Only if using PostgreSQL instead of SQLite
- **Status**: Not needed if using SQLite
